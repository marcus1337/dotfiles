# dotfiles

Personal dotfiles for Arch Linux and related setups.

## Structure

- `tmux/` — tmux config
- `nvim/` — Neovim config
- `zsh/` — Zsh config
- `vim/` — Vim config
- `install/archlinux/` — setup scripts for Arch Linux
- `sway/` — Sway config
- `fuzzel/` — Fuzzel config

## Setup

```sh
./install/archlinux/setup.sh
```

## Optional desktop tools

- `sway` — window manager / compositor
- `fuzzel` — launcher
- `dmenu` — optional launcher
